School Management System

Welcome to the School Management System! This application provides functionalities to manage student and staff data
within a school database. The application utilizes ADO.NET to communicate with the School database.

How to Use
Prerequisites

    .NET Framework installed
    Access to the School database

Setup

    Clone or download the project files to your local machine.
    Open the project in your preferred IDE (Integrated Development Environment).

Running the Application

    Navigate to the project directory and locate the Program.cs file.
    Open the file in the IDE.
    Compile and run the program.

    Upon running the program, you'll see a menu with various options:
        Option 1: View all students
        Option 2: View students from a specific class
        Option 3: Add new staff
        Option 4: View all staff
        Option 5: Get all grades from the current month
        Option 6: View mean value of grades
        Option 7: Add new students
        Option 8: Exit the application

    Choose an option by entering the corresponding number and follow the instructions displayed.

Database Connection

The program connects to a database named School. Ensure that the database exists and is accessible. 
The connection string is defined in the code files and set to connect to (localdb)\. with Windows Integrated Security.
